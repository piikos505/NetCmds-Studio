# NetCmds Studio

A simple reference for essential Packet Tracer commands, Is specifically built to support students with fast access to everything they need.

## Features

- **Complete Command Reference**: Access essential commands for PCs, routers, switches, Telnet, and SSH
- **Smart Search**: Quickly find commands with the search functionality
- **Category Filtering**: Filter commands by device type
- **Copy to Clipboard**: One-click copying of commands
- **Dark/Light Theme**: Toggle between themes for comfortable viewing
- **Interactive Chatbot**: Get help finding commands with the PiikosX Bot
- **Responsive Design**: Works perfectly on desktop and mobile devices
